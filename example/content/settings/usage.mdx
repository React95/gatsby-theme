      You can provide some configs to `@react95/gatsby-theme` theme. <br />
      They must be set in your `gatsby-config.js`, usually in project's root.

    </Frame>

Example

```javascript
// gatsby-config.js
module.exports = {
  plugins: [
    {
      resolve: '@react95/gatsby-theme',
      options: {
        // Root url for all content
        // defaults to '/'
        basePath: '/blog',
        // Location of your content,
        // defaults to 'content'
        contentPath: 'data',
      },
    },
  ],
};
```

        <strong>SEO</strong>: It's the practice of optimizing your web pages to make

Example

```javascript
// gatsby-config.js
module.exports = {
  siteMetadata: {
    // Used for the site title and SEO
    title: `Gatsby - Powered By React95`,
    // Used for SEO
    description: `This is a Gatsby website that uses React95`,
    // Used for resolving images in social cards
    siteUrl: `https://example.com`,
    // Used for SEO in social cards
    image: `/path-to/your/image`,
    // Used for SEO
    author: `@mynickname`,
  },
};
```
